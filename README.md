## 🛠 Setup Instructions

```bash
# 1. Clone the repo
git clone https://github.com/xbaelal/portfolio-new

# 2. Navigate to project directory
cd portfolio-new

# 3. Install dependencies
npm install

# 4. Run the dev server
npm run dev
```
